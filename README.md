# SpanishVerbs
スペイン語動詞検索ツール

## Get Started
### 必要なもの
 - git
 - npm

### 手順
リポジトリをクローン
```
git clone git@github.com:AtsukiImamura/SpanishVerbs.git
```

ライブラリのインストール
```
cd Spanish Verbs
npm install
```


## 開発
開発サーバの立ち上げ
```
npm run start
（止めるときは ctrl + c などで)
```

本番用ビルド
```
npm run build
```
